# Innovation Lifecycle Manager

A web application for tracking innovation initiatives from ideation through deployment.

## Purpose

This application enables companies to manage and visualize the entire lifecycle of innovation initiatives, from initial ideas submitted by employees through various maturity stages to final rollout or closure.

## Key Features

- **Kanban-Style Board**: Visual representation of initiatives across lifecycle stages
- **Idea Submission**: Any employee can submit innovation ideas
- **Lifecycle Tracking**: Track initiatives through defined maturity stages
- **Smart Search**: Advanced search with wildcards across all initiative data
- **AI Assistant**: Intelligent helper for duplicate detection, improvement suggestions, and insights
- **Analytics Dashboard**: Metrics and reporting on innovation portfolio
- **Role-Based Access**: Different permissions for employees, reviewers, and administrators
